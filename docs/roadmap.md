# Roadmap

## Phase 1 — Local functional validation
- Complete local policy/link/lab test coverage.

## Phase 2 — Real ML-KEM integration
- Replace simulated KEM paths with real implementation.

## Phase 3 — Real ML-DSA integration (if required)
- Integrate signature path and verify compatibility impacts.

## Phase 4 — Kernel-level network emulation
- Run with real `tc/netem` impairment profiles.

## Phase 5 — Real LoRa hardware campaigns
- RF field tests: loss, jitter, airtime, collisions, duty cycle.

## Phase 6 — Cryptographic audit
- Independent review of protocol changes, downgrade/replay resistance.

## Phase 7 — Hardening for private production
- Secure defaults, observability, incident playbooks, operational controls.
